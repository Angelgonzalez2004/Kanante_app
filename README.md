# Kanante App

A Flutter project.

## Description

This is a Flutter project named `kanante_app`. It uses Firebase for backend services, including authentication, database, and storage. It also includes features for handling multimedia (image picker, cached network images), maps (Google Maps, geocoding, geolocator), and other utilities.

## Features

*   Firebase integration (Core, Auth, Database, Storage)
*   Google Sign-In
*   Image picking and viewing
*   Location services with Google Maps
*   State management with Provider
*   And more...

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Flutter SDK: Make sure you have the Flutter SDK installed.
*   Firebase Project: This project requires a Firebase project to be set up.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Angelgonzalez2004/Angelgonzalez2004.git
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
